# Шашки — Русские Шашки онлайн

Браузерная игра в русские шашки с искусственным интеллектом, сохранением статистики и поддержкой светлой/тёмной темы.

## Для кого

Для всех, кто хочет сыграть в шашки прямо в браузере — как в одиночку против ИИ, так и с другом за одним устройством.

## Что умеет

- **Игра против ИИ** с тремя уровнями сложности: лёгкий, средний, сложный (алгоритм минимакс с альфа-бета отсечением)
- **Игра вдвоём** на одном устройстве с возможностью отмены хода
- **Подсказка хода** — кнопка «💡 Hint» выделяет зелёным оптимальный ход для текущего игрока (глубина поиска 3)
- **Подсветка доступных клеток** при выборе фигуры
- **Обязательные взятия** и **цепные взятия** по правилам русских шашок
- **Определение ничьей** — правило 40 ходов без взятия
- **Авторизация** — регистрация и вход по email
- **Автосохранение истории партий** — каждая сыгранная партия (победа, поражение, ничья) сохраняется автоматически
- **Статистика** — победы, поражения, текущая и лучшая серия побед
- **История ходов** текущей партии в шахматной нотации
- **Таймер** для каждого игрока
- **Адаптивный интерфейс** — работает на телефоне, планшете и десктопе
- **Светлая и тёмная тема**

## Почему это ценно

Большинство онлайн-шашек либо перегружены рекламой, либо требуют установки приложения. Здесь — чистый интерфейс, никакой рекламы, работает мгновенно в любом браузере.

## Структура базы данных

Проект использует две таблицы в Supabase с включённым Row Level Security.

### `profiles`

| Колонка | Тип | Описание |
|---|---|---|
| `id` | uuid (PK) | Совпадает с `auth.users.id` |
| `username` | text | Уникальный никнейм |
| `wins` | integer | Количество побед |
| `losses` | integer | Количество поражений |
| `best_streak` | integer | Лучшая серия побед |
| `current_streak` | integer | Текущая серия побед |
| `created_at` | timestamptz | Дата регистрации |

### `games`

| Колонка | Тип | Описание |
|---|---|---|
| `id` | uuid (PK) | Идентификатор партии |
| `player_id` | uuid (FK) | Ссылка на `profiles.id` |
| `opponent` | text | Противник (`AI` или никнейм) |
| `winner` | text | Кто победил |
| `result` | text | `win` / `loss` / `draw` |
| `game_mode` | text | `pvp` или `ai` |
| `difficulty` | text | Сложность ИИ (если режим `ai`) |
| `move_count` | integer | Количество ходов в партии |
| `duration_seconds` | integer | Продолжительность партии |
| `created_at` | timestamptz | Дата партии |

**Политики RLS:** профили доступны всем на чтение, изменять может только владелец. Партии видит и создаёт только их автор.

**Триггер `on_auth_user_created`:** при регистрации автоматически создаёт строку в `profiles`, читая никнейм из метаданных пользователя.

## Ограничения (Supabase Free Tier)

Проект работает на **бесплатном тарифе Supabase**, у которого есть ряд ограничений:

- **Лимит отправки email** — не более **2 писем в час** на весь проект. Если несколько пользователей пытаются зарегистрироваться подряд, Supabase вернёт ошибку `429 email rate limit exceeded`. Решение: подождать час или подключить собственный SMTP (SendGrid, Resend и т.д.) в настройках проекта.
- **Пауза проекта** — неактивные проекты на Free Tier автоматически приостанавливаются через 7 дней без обращений. Первый запрос после паузы может занять несколько секунд.
- **Лимиты БД** — 500 МБ хранилища, 2 ГБ исходящего трафика в месяц.
